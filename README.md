# Python-Programming ( Smart Profile Gen. )
Task 1 : Smart Profile Generator Using Python 


How it Works

•	Variables: Think of these as labeled boxes (like name or age) that hold the information the user types in.

•	Input(): This function stops the program and lets the user type. Whatever they type gets saved into a variable.

•	Int(): By default, Python sees everything you type as "text" (a string). We use int() to tell Python that the age should be treated as a whole number.

•	f-strings: Putting an f before a string (like name: {name}") allows us to put variables directly inside the text using curly braces {}.
